# certbot
AI Chatbot to help answer questions for volunteers in CERT (Community Emergency Response Team)

This is to explain the purpose, functionality, and implementation of CERTBOT. CERTBOT is an AI chatbot meant to provide timely and accurate answers to questions volunteers have about the CERT program. Primarily sourced from the FEMA CERT website and related documentation about the national CERT program.

This responsitory is to support this by explaining how anyone can create and deploy their own CERTBOT by following these steps.
